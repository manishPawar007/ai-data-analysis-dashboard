# AI Powered Data Analysis Dashboard

## Features
- Upload CSV, Excel and JSON files
- Data Cleaning
- Interactive Charts
- SQL Analysis
- Machine Learning Predictions
- PDF, CSV and Excel Reports

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Seaborn
- Scikit-Learn
- SQLite

## Run Locally

pip install -r requirements.txt
streamlit run app.py
